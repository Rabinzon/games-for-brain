### Quick start:

```sh
$ npm i -g games-for-brain
$ brain-even
```

### Available games:
- brain-even
- brain-calc
- brain-gcd
- brain-prime
- brain-balance
- brain-progression

### Running in dev mode
```sh
$ make install start game=$(game-name)
```
